# 🏠 Автоматизации для Home Assistant

Коллекция готовых автоматизаций для Home Assistant, которыми я хочу поделиться с сообществом.

## 📋 Что здесь есть

В этом репозитории собраны различные автоматизации для умного дома — от простых сценариев освещения до комплексных систем управления. Каждая автоматизация проверена и работает в реальных условиях.

## 🗂 Структура репозитория

```
📁 automations/
   ├── 📁 lights/         — автоматизации освещения
   ├── 📁 climate/        — управление климатом
   ├── 📁 security/       — безопасность и оповещения
   ├── 📁 media/          — мультимедиа сценарии
   └── 📁 other/          — прочие автоматизации
📁 blueprints/            — готовые чертежи (blueprints)
📁 scripts/               — полезные скрипты и шаблоны
```

## 🚀 Как использовать

1. Выбери нужную автоматизацию из папок
2. Скопируй содержимое файла YAML
3. Добавь в свой `automations.yaml` или через UI редактор Home Assistant
4. Настрой сущности (entity_id) под своё оборудование
5. Перезагрузи автоматизации

> ⚠️ Перед использованием **проверь названия устройств и сенсоров** — они должны соответствовать твоей конфигурации.
